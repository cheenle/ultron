# ULTRON Python版本依赖
# 所有依赖都是Python标准库，无需额外安装

# 使用的标准库模块:
# - socket: UDP网络通信
# - json: JSON数据解析
# - time: 时间处理
# - datetime: 日期时间处理
# - re: 正则表达式
# - os: 操作系统接口
# - sys: 系统相关功能
# - threading: 多线程支持
# - pathlib: 路径操作
# - dataclasses: 数据类
# - typing: 类型提示
# - argparse: 命令行参数解析
# - importlib: 动态导入模块

# 最低Python版本要求: 3.7+
# 推荐Python版本: 3.9+ (更好的类型提示支持)
